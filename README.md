# Задача A3b из SET 3 по предмету "Алгоритмы и структуры данных"
В данном репозитории представлены исходные данные эксперимента, реализация классов ArrayGenerator и SortTester, а также графики, использованные в отчёте. 

### Навигация по репозиторию:
- arraygenerator.h - реализация класса ArrayGenerator.
- sorttester.h - реализация класса SortTester.
- introsort - в папке data представлены исходные данные и графики, в папке graphics - графики.
- quicksort - то же самое, но для quick sort.
- A3b_SET3.pdf - отчёт по задаче с выводом.
